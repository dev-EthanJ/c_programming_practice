# 독하게 시작하는 C 프로그래밍
## Chapter 16. 전처리기
### 연습문제

1. 다음 매크로 정의의 문제점은 무엇인가?

```c
#define ADD(a, b) a + b
```

- `a + b`를 괄호로 묶지 않아서, `ADD()` 파라미터 내부에서나 아니면 해당 매크로가 포함된 코드의 연산 우선순위 문제가 발생한다.
- `ADD(a, b) ((a) + (b))`로 작성해야 한다.

<br>

2. `__inline` 함수와 매크로의 공통점은 무엇인지 설명하세요. 

- 자주 사용되는 코드를 함수 오버헤드(호출/복귀, 스택 프레임 생성)를 줄여서 사용할 수 있다.
