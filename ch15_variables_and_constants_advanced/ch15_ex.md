# 독하게 시작하는 C 프로그래밍
## Chapter 15. 변수와 상수 고급 이론
### 연습문제

1. `const int *data`와 `int* const data`의 선언은 모두 `int`형에 대한 포인터 변수 선언입니다. 그러나 `const` 형한정어가 적용된 대상이 다릅니다.
구체적으로 무엇이 어떻게 다른지 기술하세요.

- `const int *data`는 `data` 포인터가 가리키는 대상인 `int`형 자료를 상수화한다. 따라서 해당 자료에 대해 접근해서 읽기는 가능하지만 `*data = 10;`등의 값 수정은 불가하다.

- `int* const data`는 `data` 포인터 자체를 상수화했다. 따라서 해당 포인터가 가리키는 자료를 읽고 수정 모두 가능하지만, `data = NULL;`등의 `data` 포인터 자체를 수정할 순 없다.

2. 심볼릭 상수(symbolic constant)를 정의할 수 있는 두 가지 방법이 무엇인지 답하세요.

- 첫번째는 `#define` 전처리기를 이용하는 것이고, 두번째는 `const` 형한정어를 사용할 수 있다. 프로그램 코드 내에서 의미있는 상수값을 지정할 때 사용할 수 있다.

3. 외부 변수를 선언하기 위한 예약어는 무엇인지 답하세요.

- 현재 코드 파일의 외부에 정의가 존재하는 변수에 대해서 `extern` 예약어를 사용해서 선언하고 사용할 수 있다.

4. 게임을 개발하면서 상단, 바닥, 왼쪽, 오른쪽에 해당하는 정보를 열거형 상수로 정의하려 합니다. 만일 열거형 상수의 이름을 `MYMOVE`라고 하고
속한 값으로 `TOP, BOTTOM, LEFT, RIGHT`를 선언하려 합니다. 코드를 작성하세요.

```c
#include <stdio.h>
typedef enum MYMOVE { TOP, BOTTOM, LEFT, RIGHT } MYMOVE;
```
