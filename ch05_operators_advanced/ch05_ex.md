# 독하게 시작하는 C 프로그래밍
## Chapter 05. 연산자 - 응용
### 연습문제
1. 다음 코드의 실행 결과 및 원리에 대해 답하세요.
```c
#include <stdio.h>

int main(void)
{
    int nResult, w = 0, x = -1, y = -1, z = 1;
    nResult = w++ || x++ && ++y || ++z;
    printf("%d %d %d %d %d\n", w, x, y, z, nResult);

    return 0;
}
```
- `nResult = w++ || x++ && ++y || ++z;` 코드에서 연산 실행 순서와 각 변수의 값에 주목해야 한다. 가장 먼저 실행되는건, `x++ && ++y` 이다. 이때에 `x++`은 해당 논리곱 연산이 실행된 후에 증가하는 연산이므로, 실제 논리곱은 `-1 && 0`이다. 따라서 결과는 `0`이다. 이 이후 `x==0`이고, `y==0`이다.
- 그 다음, `w++ || 0 || ++z` 연산이 수행된다. 왼쪽부터 차례대로 실행되어서, 먼저 `w++ || 0`연산이 실행된다. 이 또한 `0 || 0`이므로 결과는 0이다. 이때 `w==1`이 된다. 이후 `0 || ++z`연산에서, `0 || 2`이므로 결과는 `1`이고 `z==2`이다.
- 마지막으로 `nResult = 1`이 실행되며 변수에 대입된다
- 따라서 최종적으로 콘솔에 표시되는 내용은, `1 0 0 2 1`이다.
