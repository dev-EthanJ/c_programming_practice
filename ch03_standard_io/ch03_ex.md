# 독하게 시작하는 C프로그래밍
## Chapter 03. 표준 입출력 도구

### 연습문제
1. `getchar()`함수를 호출하여 사용자로부터 영문 네 글자 이상을 입력받고, 그중에서 두 번째 글자만 화면에 출력하려고 합니다. 프로그램 코드를 작성하세요.

```c
#include <stdio.h>

int main(void)
{
    char ch = 0;
    
    ch = getchar();
    ch = getchar();

    putchar(ch);

    return 0;
}
```

2. `getchar()`함수와 `_getch()`함수가 내부적으로 다른 점은 무엇인지 기술하세요.
- `getchar()`함수는 stdin 버퍼를 사용하고, `_getch()`함수는 버퍼가 존재하지 않는다.


3. `gets()`함수는 보안결함이 있습니다. 이 함수를 대체할 수 있는 함수에는 어떤것이 있는지 기술하세요.
- `gets_s()`함수를 사용하거나, `scanf()`함수를 `%s`형식문자와 함께 사용한다.

4. 만일 부호가 있는 10진 정수 형식으로 표현할 수 있는 가장 큰 양수의 범위를 1만큼 넘어서면 컴퓨터는 이를 어떤 숫자로 인식하는지 기술하세요.
- 부호비트가 1로 바뀌고 data bit는 모두 0으로 바뀐다. 이를 2의 보수로 계산하여 결국 가장 큰 음수의 숫자로 인식된다.

5. 부호가 없는 10진 정수 형식과 부호가 있는 10진 정수 형식을 출력하기에 적절한 형식 문자는 각각 무엇인지 답하세요.
- 부호 없는 10진 정수는 `%u`를 사용하고, 부호있는 10진 정수는 `%d`를 사용한다.

6. `123.45`의 자료형은 무엇이며 이 값을 출력하기에 적절한 형식 문자는 무엇인지 답하세요.
- double 자료형이며 `%f` 또는 `%lf`를 사용한다.

7. 형식문자 `%012.3F`가 의미하는 것은 무엇인지 기술하세요.
- 총 12자리로 표현된 실수, 오른쪽 정렬 출력, 빈자리 0으로 채움, 소수점 아래 3번째 자리까지 반올림해서 표시.

8. 배열에 담긴 문자열을 출력하려 합니다. 적절한 형식문자는 무엇인지 답하세요
- `%s`

9. `scanf()`함수로 정수를 입력받은 후, 입력버퍼에 남은 개행문자를 제거하려 합니다. 어떤 방법이 있는지 기술하세요.
- `scanf()`함수의 형식문자열에, `%d`뒤에 개행문자를 입력받고 버릴 수 있는 `%*c` 형식문자를 추가한다. 
- 또는 `getchar()`를 사용하여 개행문자를 제거한다.

10.  만일 `nInput`이 정수형 변수라면 다음 코드는 무엇이 잘못되었는지 기술하세요.
```c
scanf("%d",nInput);
```
- scanf()함수로 입력받은 값을 저장하는 변수의 주소를 parameter로 전달해야 한다. 따라서 `&nInput`으로 수정이 필요하다.


11. `scanf()`함수로 double형 실수를 입력받으려 합니다. 적절한 형식문자는 무엇인지 답하세요.
- `%f`
