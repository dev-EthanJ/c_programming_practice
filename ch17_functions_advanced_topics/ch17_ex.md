## Chapter 17. 함수에 대한 고급 이론
### 연습문제

1. C/C++ 언어에서 사용하는 기본 함수 호출 규약을 쓰세요.

- C언어는 `__cdecl`를 사용하고, C++은 `__cdecl` (일반 함수), `__thiscall` (멤버 함수)를 사용한다.

<br><br>

2. 원형이 `char* __stdcall TestFunc*(int, double);`인 함수에 대한 포인터 변수를 선언하려고 합니다. 적절한 자료 형식을 기술하세요.

```c
char* (__stdcall *FuncPointer) (int, double);
```

<br><br>

3. 특정 함수를 호출할 때 함수의 주소를 매개변수로 전달한 후, 주소가 전달된 함수가 특정함수로부터 호출되는 구조는 무엇인지 쓰세요.

- 역호출 구조, call back

<br><br>

4. 정적 라이브러리 파일의 확장명은 무엇인지 쓰세요.

- Window에서는 `.lib`, Linux/Mac에서는 `.a`

<br><br>

5. 헤더파일이 여러 번 중복해서 포함되는 것을 막기 위한 전처리기 코드는 무엇이고, 이 코드를 기술하는 위치는 헤더파일 내부중 어디인지 쓰세요

- `#pragma once` 코드를 헤더파일 최상단에 기재한다.  
```c
#ifndef 헤더파일상수이름
#define 헤더파일상수이름
// 헤더 파일 내용, 변수 및 함수 선언
#endif  
```
- 또는 위 코드를 사용할 수 있다.
