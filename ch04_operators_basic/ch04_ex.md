# 독하게 시작하는 C 프로그래밍
## Chapter 04. 연산자 - 기본
### 연습문제

1. 연산의 중첩에서 연산의 '임시결과'란 무엇을 의미하며, 임시결과의 가장 중요한 특징은 무엇인지 기술하세요.
- 임시결과란, 해당 구문의 다음 연산이 수행되기 이전 수행된 연산의 임시 결과를 의미하며, 해당 임시결과는 별도로 저장하지 않으면 삭제된다는 특징이 있다.

2. char형 자료와 int형 자료간에 `+`연산을 수행하면 임시결과의 자료형은 무엇인지 답하세요.
- type promotion이 발생해서 int형 자료로 결과가 나온다.

3. 이형자료간의 연산에서 임시결과에 대한 자료형이 결정되는 근거는 무엇인지 기술하세요.
- type promotion에서 더 메모리 사이즈가 큰 자료형으로 결정된다.

4. `5 / 2`의 연산 결과의 자료형식과 값은 무엇입니까?
- int형 정수 2가 결과값이다.

5. 정수형 `nInput`변수를 5로나눈 나머지를 출력하는 구문을 작성하세요.
`printf("%d\n", nInput % 5);`

6. 대입 연산자의 왼쪽 피연산자가 될 수 있는 자격을 갖춘 대상을 부르는 용어는 무엇인지 답하세요.
- l-value (left value & locating value)

7. 다음 코드의 실행결과는 무엇인지 작성하세요.
```c
#include <stdio.h>
int main(void)
{
    printf("%08x\n", 0xFFFFFFFF & 0x00080000);
    printf("%08x\n", 0xAAAABBBB & 0x00080000);
    printf("%08x\n", 0xAABBCCDD & 0x00080000);

    printf("%08x\n", 0xFFFFFFFF | 0x00080000);
    printf("%08x\n", 0x11223344 | 0x00080000);
    printf("%08x\n", 0x00000000 | 0x000F0000);

    return 0;
}
```
